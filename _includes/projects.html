SELECTED PROJECTS

Brain Connectivity

website

→ Developed an interactive platform for brain connectivity learning, analysis and visualization. Written with R and Shiny. Hosted in AWS.

→ Used sparse inverse covariance estimation to model high dimensional brain connectivity network, which is more interpretable and robust than existing brain network modeling methods.

→ D3 JavaScript library was embedded for interactive visualization.

Insight

Automatic Data Question Answering; website

→ Collaboratively working with data scientists from LinkedIn, to develop an comprehensive data mining web-based platform, to help people analyze their datasets automatically.

→ From machine learning aspect, common regression, classification, clustering and text mining methods were used to analyze the datasets of different types.

→ Artificial intelligence component: given plain text questions by users, word2vec was used to extract the intention of users, and automatically feed best answers back to users.

→ From technical aspect, INSIGHT project was mainly written with Go, R and SQL.
